# Neon Synth Snake 🐍 - Retro Arcade Game

Neon Synth Snake is a browser-based arcade game inspired by the classic Snake, redesigned with a futuristic neon / synthwave aesthetic.

# Features
Classic snake gameplay mechanics
Neon-themed visuals and animations
Real-time keyboard controls
Smooth game loop and rendering
Responsive browser gameplay

# Technologies Used
ClojureScript
re-frame
HTML
CSS

# Project Objective
To build an interactive and visually engaging version of the classic Snake game while demonstrating functional programming concepts and modern frontend architecture using ClojureScript.
